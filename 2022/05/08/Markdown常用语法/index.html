Markdown常用语法

发表于2022-05-08|更新于2022-05-15|Tools
|阅读量:

使用Typora编辑器

基本语法格式

名称 语法 快捷键
标题 用#表示，#一级标题，##表示二级标题，以此类推 Ctrl + 1、2、3、4
斜体字 左右用 * 包着 Ctrl + I
字体加粗 左右用 ** 包着 Ctrl + B
加粗斜体 左右用 *** 包着 Ctrl + I ， Ctrl + B（无先后顺序）
中划线 左右用 ~~ 包着 Alt + Shift + 5
引用 文字开头加上 >
链接 [链接名称](链接网址) Ctrl + K
代码块 开头结尾用```包着 Ctrl + Shift + K
公式块 行内公式：左右用$包上；行间公式：开头结尾用$$包上 Ctrl + Shift+ M（行间公式）
表格 首行用 | 分隔（表格内换行需要用<br>） Ctrl + T
Ctrl + Enter 添加行

数学公式部分

名称 语法
分数 \frac{分子}{分母}
次幂 a^b（幂数可以用大括号包起来）
分段函数 函数名=\begin{cases}
公式1 & 条件1
公式2 & 条件2
\end{cases}
小于等于号 \leqslant
大于等于号 \geqslant

转译字符

在 Markdown 中使用 \ 无法转译的字符需要使用字符的命名实体或十进制编码，再加上分号;

特殊符号 命名实体 十进制编码
竖线 &#x7C &#124
< &lt &#60
> &gt &#62
{ &#123
} &#125
\ &#92
